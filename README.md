##Zetta FONA GPRS serial device driver

###Install

```
$> npm install zetta-fona-gprs-serial-driver
```

###Usage

```
var zetta = require('zetta');
var FonaGPRS = require('zetta-fona-gprs-serial-driver');

zetta()
  .use(FonaGPRS)
  .listen(1337)
```
